# 에이커머스 포터(Porter) 프로젝트 분석 보고서

## 1. 프로젝트 개요
본 프로젝트는 "포터(Porter)" 및 "포트(Potr)" 브랜드를 위한 React 기반 이커머스 플랫폼입니다. 현대적이고 반응형인 UI를 갖추고 있으며, 상품 탐색, 장바구니 관리, 결제 프로세스, 사용자 인증, 커뮤니티 게시판 등 이커머스의 핵심 기능을 포괄적으로 구현하고 있습니다.

## 2. 기술 스택
- **프레임워크:** React 19
- **빌드 도구:** Vite
- **상태 관리:** Zustand (Persist 미들웨어 사용)
- **라우팅:** React Router Dom v6
- **스타일링:** Sass (SCSS)
- **애니메이션:** GSAP, Swiper
- **아이콘:** Lucide React
- **모달/알림:** Sweetalert2
- **기타 라이브러리:** React Daum Postcode (주소 검색), React Quill (리치 텍스트 에디터)

## 3. 아키텍처 및 구조

### 3.1 디렉토리 구조
- `src/assets/api`: 상품 및 게시판 데이터를 위한 모의(Mock) 데이터 포함.
- `src/components`:
  - `common`: `Header`, `Footer`와 같은 공통 컴포넌트.
  - `main`: 홈 화면 전용 섹션들 (`Hero`, `BrandStory`, `PotrSection`).
  - `Intro`, `ProductItem`: 재사용 가능한 UI 컴포넌트.
- `src/pages`: 개별 페이지 컴포넌트 (메인, 상품 목록/상세, 장바구니, 결제, 마이페이지, 게시판 등).
- `src/store`: 모듈형 상태 관리를 위한 Zustand 스토어 및 슬라이스 설정.
- `src/styles`: 글로벌 SCSS 변수 및 믹스인(Mixins).

### 3.2 상태 관리 (Zustand)
Zustand를 사용하여 중앙 집중식 스토어를 운영하며, 기능별로 슬라이스를 분리하여 관리합니다.
- `authSlice`: 회원가입, 로그인/로그아웃, 프로필 업데이트 처리.
- `productSlice`: 상품 데이터 관리, 필터링 로직 및 리뷰 관리.
- `cartSlice`: 장바구니 품목, 수량 조절 및 금액 계산.
- `orderSlice`: 주문 내역 저장 및 관리.
- `boardSlice`: 공지사항, FAQ, 1:1 문의 및 Q&A 관리.
- `wishlistSlice`: 사용자의 관심 상품 관리.

**데이터 유지(Persistence):** 스토어의 주요 데이터는 `localStorage`(`ff-storage` 키)에 저장되어 브라우저를 새로고침해도 유지됩니다.

## 4. 주요 기능

### 4.1 상품 탐색 및 상세
- **상품 목록:** 카테고리(Work, Daily, Travel 등), 성별(Men, Women, Kids), 특수 태그(Best, Sale)별 필터링 기능.
- **검색:** 상품 목록 페이지와 연동된 실시간 검색 기능.
- **상품 상세:** 상세 스펙 확인, 컬러/사이즈/수량 선택, 관련 상품 추천 기능.

### 4.2 장바구니 및 결제
- **장바구니:** 개별/전체 선택, 수량 변경, 총 결제 금액 계산 (100,000₩ 이상 무료 배송 로직 포함).
- **결제(Checkout):** 회원 및 비회원 결제 지원. Daum 주소 API를 통한 배송지 입력 기능.

### 4.3 마이페이지 (회원 전용)
- **프로필 관리:** 연락처 및 배송지 정보 수정.
- **주문 관리:** 과거 주문 내역 및 배송 상태 조회.
- **관심 상품:** 위시리스트에 담긴 상품 빠른 확인.
- **활동 관리:** 본인이 작성한 Q&A, 1:1 문의, 상품 리뷰의 조회 및 수정/삭제.

### 4.4 고객 지원 및 커뮤니티
- **게시판:** 공지사항 및 카테고리별 FAQ (아코디언 UI).
- **1:1 문의:** React Quill을 이용한 리치 텍스트 기반 문의 작성 및 마이페이지 연동.
- **리뷰 시스템:** 상품 상세 페이지 내 별점 및 텍스트 리뷰 등록 기능.

## 5. UI/UX 특장점
- **인트로 애니메이션:** `Intro.jsx`를 통한 감각적인 진입 화면 제공.
- **스티키 헤더:** 스크롤 위치에 따라 반응하는 헤더 레이아웃 (`Main.jsx` 참조).
- **스크롤 복구:** 페이지 이동 시 화면 최상단으로 자동 이동 및 수동 복구 로직 적용.
- **인터랙티브 피드백:** `Sweetalert2`를 활용한 세련된 토스트 알림 및 확인창 제공.

## 6. 개발 현황 및 확장성
현재 프로젝트는 모의 데이터(`productData.js`, `boardData.js`)를 사용하여 모든 이커머스 로직이 완벽하게 구현된 상태입니다. 구조적으로 백엔드 API와의 연결이 용이하도록 슬라이스 기반의 상태 관리가 잘 구축되어 있으며, 실제 서비스 전환 시 API 호출 부분만 교체하면 즉시 운영이 가능한 수준의 완성도를 보이고 있습니다.
